# Changelog

All notable changes to Privacy Guard will be documented in this file.

## [2.0.0] - 2024-XX-XX

### Added
- Complete rewrite as "Privacy Guard" from "T&C Buddy"
- Multi-mode analysis system (AI, Local, Hybrid)
- Support for multiple AI providers (Deepseek, OpenAI, Anthropic, Gemini)
- Enhanced red flag detection (20+ categories)
- Cookie and tracker analysis
- Cookie consent banner detection and auto-decline
- Tracker blocking system
- Privacy score calculation (0-100)
- Readability score
- Tabbed popup interface (Summary, Details, Cookies, History, Settings)
- Options page for full configuration
- Analysis history tracking
- Export functionality
- Comprehensive documentation

### Changed
- Upgraded to Manifest V3 architecture
- Improved UI/UX with modern design
- Enhanced analysis accuracy
- Better error handling

### Fixed
- Various bug fixes and improvements

## [1.0.0] - 2024-XX-XX

### Added
- Initial release as "T&C Buddy"
- Basic Terms & Conditions analysis
- Deepseek AI integration
- Local heuristic fallback
- Simple popup interface
- Red flag detection (8 categories)
